# Tickr | Stock Tracker

A full-stack web application for tracking SWE stocks with end-of-day market data. Built with Next.js, Express.js, and Docker.


## Built with
- Node.js 20+
- Docker and Docker Compose
- MongoDB database
- Alpha Vantage API 
- Redis
